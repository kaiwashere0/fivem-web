<h2><i class="fa-solid fa-circle-info"></i> Genel Bakış</h2>

<p>
<b>Kai Web Management</b>, FiveM sunucularının Discord tarafını
<b>web tabanlı, merkezi ve güvenli</b> bir panel üzerinden yönetmek için
geliştirilmiş <b>lisanslı ve ücretli</b> bir yazılımdır.
</p>

<p>
Rütbelendirme, rol izinleri, whitelist ve yetkili başvuru sistemleri gibi
kritik yönetim süreçlerini <b>tam kontrol, kayıt ve yetkilendirme</b> mantığıyla sunar.
</p>

<blockquote>
<b>Hedef:</b> 35 GitHub Star<br/>
Projeyi desteklemek için star bırakabilirsiniz.
</blockquote>

<h2><i class="fa-solid fa-layer-group"></i> Sistem Özellikleri</h2>

<h3><i class="fa-solid fa-globe"></i> Web Yönetim Paneli</h3>
<ul>
  <li>Web tabanlı yönetim paneli</li>
  <li>Yetkiye bağlı erişim kontrolü</li>
  <li>Modern ve sade arayüz (Liquid Glass tasarım yaklaşımı)</li>
  <li>Gerçek zamanlı veri yönetimi</li>
</ul>

<h3><i class="fa-solid fa-id-badge"></i> Rütbe & Rol Yönetimi</h3>
<ul>
  <li>FiveM sunucularına özel rütbelendirme sistemi</li>
  <li>Discord rol atama ve kaldırma</li>
  <li>Rol bazlı yetki sınırlandırmaları</li>
  <li>Yetkili hiyerarşi kontrolü</li>
</ul>

<h3><i class="fa-solid fa-clipboard-list"></i> Whitelist Sistemi</h3>
<ul>
  <li>Oyuncular için whitelist başvuru sistemi</li>
  <li>Yetkililer için onay / red paneli</li>
  <li>Başvuru durum takibi</li>
  <li>Başvuruların arşivlenmesi</li>
</ul>

<h3><i class="fa-solid fa-user-shield"></i> Yetkili Başvuru Yönetimi</h3>
<ul>
  <li>Yetkili başvuru formları</li>
  <li>Rütbeye göre değerlendirme sistemi</li>
  <li>Yetki verme ve geri alma işlemleri</li>
  <li>Yetkili işlem kayıtları</li>
</ul>

<h3><i class="fa-solid fa-file-shield"></i> Loglama & Kayıt Sistemi</h3>
<ul>
  <li>Tüm işlemler otomatik olarak loglanır</li>
  <li>Kullanıcı, tarih ve işlem bilgisi</li>
  <li>Web panel üzerinden detaylı log görüntüleme</li>
</ul>

<h2><i class="fa-solid fa-gear"></i> Kurulum</h2>

<pre>
git clone https://github.com/kaiwashere0/KaiWebManagement.git
cd KaiWebManagement
npm install
npm start
</pre>

<p>
<b>Not:</b> <code>.env</code> dosyası Discord bot tokeni ve FiveM yapılandırmaları ile doldurulmalıdır.
</p>
